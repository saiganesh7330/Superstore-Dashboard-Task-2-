# Task 2 – Data Visualization & Storytelling

## Objective
Create visualizations to convey business insights using a Superstore sales dataset.

## Tools
- Power BI Desktop

## Files
- Superstore.csv
- Task2_Superstore_Dashboard.pdf

## Visuals included
1. KPI cards (Total Sales, Total Profit, Total Quantity)
2. Monthly Sales Trend (Line chart)
3. Sales by Region (Bar chart)
4. Sales by Category (Bar chart)
5. Profit by Sub-Category (Column chart)
6. Top 10 Customers (Horizontal bar)
7. Summary page with key insights

## Key Insights (from dataset)
- Total Sales: ₹200,451.61
- Total Profit: ₹7,926.22
- Top Region: South (₹58,966.10)
- Top Category: Furniture (₹72,911.82)
- Worst Sub-Category (by profit): Storage
- Top Month: July 2022 (₹24,619.49)

## How to reproduce
1. Open Power BI Desktop → Get Data → Text/CSV → load Superstore.csv.
2. Follow the visuals creation and formatting steps in the report.
3. Export the report to PDF (File → Export → Export to PDF).
